这可能是因为 Markdown 语法对“复合表头”的支持比较有限。在标准的 Markdown 表格中，第一行会被强制识别为表头，因此很难直接做出那种带“嵌套层级”的大表头。

你无法粘贴  和  可能是因为你直接复制了含有 LaTeX 数学公式的代码，而你的编辑器环境可能不支持实时渲染。

为了实现你要求的 **“ 作为大表头”** 效果，通常建议采用以下两种布局方式：

### 方案一：标准 MD 兼容版（最推荐）

这种方式将  放在首行作为总览，结构最稳定。

### 方案二：如果你是在支持 HTML 的环境下（如 Notion, Obsidian, GitHub 等）

可以使用 HTML 标签来实现**真正的合并单元格**，这能让  真正垂直居中或跨列显示。

```html
<table>
  <thead>
    <tr>
      <th rowspan="2">能力权重 (β)</th>
```
